规章制度

1．文学院图书资料中心522室――524室为开架借阅，读者可根据需要自行取书。请保持书架图书的排列顺序，不借的图书可放在阅览车上，不可随意乱放。

2．526室（艺术书室）为闭架借阅，读者可在检索机器上查找所需图书分类号，交与值班人员，统一取书。

3．请读者自觉爱护公共图书，不要涂抹、污损图书。

4．硕士生、博士生凭本人阅览证可借阅5册图书，借期一个月，请按期归还，逾期需缴纳罚款0.10元/天。图书可续借一次，续借手续请携带本人借阅证和图书到图书资料中心办理。

5．请勿携带食品、饮料进入图书资料中心，并保持桌面、书箱整洁。

新生指南

1.本院学生凭本人阅览证进入图书资料中心，在登记册上登记姓名、证号。阅览证仅限本人使用，发现冒名者一律没收。

2.带包的读者凭阅览证换取存包柜钥匙并存包（贵重物品自理），请勿携带书籍进入阅览室。

3.每人每次限取阅2册图书到座位上阅览，阅览后将图书放在阅览车上。
